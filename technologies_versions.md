# Technologies and Versions

| Component | Technology | Version / Model | Purpose |
| :--- | :--- | :--- | :--- |
| **Backend Framework** | FastAPI | `fastapi==0.111.0`, `uvicorn==0.29.0` | High-performance Python web framework for the API. |
| **Machine Learning (NLP)** | Hugging Face Transformers | `transformers==4.40.1`, `torch==2.3.0` | Provides the pre-trained BERT model. |
| **Sentiment Model** | BERT | `nlptown/bert-base-multilingual-uncased-sentiment` | 1-5 star sentiment rating model. |
| **Generative AI (LLM)** | Gemini API | `google-generativeai==0.5.2` (Gemini Pro) | Generates the dynamic linguistic synthesis paragraph. |
| **Frontend Framework** | HTML/JS/CSS | Vanilla | Lightweight, custom UI design without heavy frameworks. |
| **Data Visualization** | Chart.js | `v4.x` (CDN) | Renders interactive, animated charts on the frontend. |
| **Containerization** | Docker | `latest` | Packages the application for consistent production deployment. |
| **Environment Management**| Python Dotenv | `python-dotenv==1.0.1` | Loads environment variables (like API keys) safely. |
